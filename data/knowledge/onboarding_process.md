# 新生报到与入学流程

本文件依据本地《新生手册》整理，用于回答“报到流程、入学流程、开学要做什么、迎新怎么走”等问题。固定日期、具体地点、费用金额和学院安排每年会变化，应以迎新网、交我办 App 和学院最新通知为准。

## 到校前

1. 查看学校迎新网、录取通知书随附材料和学院通知，确认报到时间、校区、学院集合点、宿舍安排查询时间及交通要求。
2. 注册并激活 jAccount。手册说明可通过迎新网或网络信息中心主页进入注册入口，注册时需填写姓名、证件号码、录取通知书编号等信息。
3. 登录“交我办 App”，进入“新生-我的入学清单”，按要求完成思源码领取、信息采集、新生宿舍需求调研、学费缴纳、健康状态调研等业务。
4. 按学校缴费指南通过官方渠道完成缴费。家庭经济困难、暂时无法完成缴费的新生，可按通知申请“绿色通道”暂缓缴费入学。
5. 准备录取通知书、身份证件、照片、党团组织关系、户口迁移、档案、资助贷款等材料。不同学生类型所需材料不同，应以当年通知为准。
6. 查询宿舍安排，准备必要生活用品。手册提示到校急需的生活用品和报到材料应随身携带，避免行李托运或快递延迟影响报到。

## 报到当天

1. 到达校区后先前往宿舍楼或学院指定报到点。2025 级本科手册中说明，本科新生入住宿舍时可根据志愿者引导在宿舍楼办理报到注册手续。
2. 持录取通知书和身份证件完成身份核验和报到注册。
3. 到宿舍楼办理入住，领取宿舍钥匙或门禁权限，检查宿舍设施和代办生活用品。如发现缺失或损坏，及时联系宿舍管理员。
4. 关注学院、班级、辅导员、班主任或临时召集人的通知，确认入学教育、体检、军训、班会、选课说明等后续安排。
5. 如报到过程中遇到困难，可到所属院系教务办公室、学院迎新点或学校学生事务相关窗口咨询。

## 入学后一周

1. 熟悉宿舍、食堂、图书馆、校医院、学院楼、教学楼、快递点等常用地点。
2. 按通知办理党团组织关系、户口迁移、档案移交、医保和体检等后续事项。
3. 保存辅导员、班助、学院教务、宿管、校医院等官方联系方式。
4. 查看课表和教学楼位置，提前走一遍常用路线。
5. 关注网络、校园卡、思源码、门禁和邮箱是否可正常使用，异常时通过官方渠道报修或咨询。

## 常见降级回答

如果用户只问“报到当天怎么办”，可以回答：先带好录取通知书和身份证件，到宿舍楼或学院指定地点完成报到注册，再办理宿舍入住，随后确认学院通知、体检、入学教育和后续手续。具体地点和时间以当年迎新网和学院通知为准。
